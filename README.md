# DemiCSS
Simple and Lightweight CSS Designs
```html
<button class="demi-btn demi-primary">Button</button>
```

# Installation
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/dlvdls18/DemiCSS@main/src/demi.css">
```
