Coloring

Demi CSS have 7 colors you can use.


Name Color
Primary #028DD1
Secondary #1F1F1F
Forest #377939
Crimson #8F312E
Gold #A69B25
Hollow #6E6E6E
Light #D2D2D2

Buttons

DemiCSS have buttons that only available as outline.

Add the class demi-btn for the button style, and the class demi-btn-{color} for its color.


Inputs

DemiCSS have buttons that only available as filled.

Add the class demi-input for the button style, and the class demi-input-{color} for its color.
