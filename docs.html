Buttons

DemiCSS have buttons that only available as outline.

Use the class demi-btn for the button style, and the class demi-btn-{color} for its color.


Inputs

DemiCSS have text inputs that only available as filled.

Use the class demi-input for the button style, and the class demi-input-{color} for its color.


Sliders

DemiCSS have range inputs which is called a slider.

Use the class demi-slider for the button style, and the class demi-slider-{color} for its color.
